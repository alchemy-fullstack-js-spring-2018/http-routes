# How To Use
## GET requests to /books
* `/books` 
    * returns **all books in db**

* `books/id`
    * returns **book with that id**

## POST request to /books
* adds book to db

## PUT request to /books
* updates **book with that id**

## DELETE request to /books
* deletes **book with that id**
